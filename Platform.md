## One Platform to Rule Them All

Modern **.NET** makes development simple by providing **one platform** to build **many types of applications**.  
With the same language (**C#**), same tools, and same libraries, developers can work across different technologies.

###  Supported Workloads

- **Cloud Native**  
  Microservices, Azure Functions, Containers

- **Web Full-Stack**  
  Blazor, ASP.NET Core, Web APIs, WASM

- **Cross-Platform UI**  
  .NET MAUI (iOS, Android, macOS, Windows)

- **AI & Machine Learning**  
  ML.NET for building and consuming ML models

-  **Game Development**  
  Unity with C#

- **IoT & Embedded Systems**  
  Lightweight and high-performance runtimes
