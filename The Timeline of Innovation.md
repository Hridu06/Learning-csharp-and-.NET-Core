## Evolution Overview

| Year | Version | Milestone | Key Impact |
|----|-------|----------|-----------|
| 2002 | C# 1.0 | Beginning | Managed Code, CLR |
| 2007 | C# 3.0 | Data Revolution | LINQ |
| 2012 | C# 5.0 | Async Era | async / await |
| 2016 | .NET Core 1.0 | Cross-Platform | Windows, Linux, Mac |
| 2020+ | .NET 5–8 | Unification | One Unified .NET |

---

## 2002 – The Beginning  
### C# 1.0 & .NET Framework

-  C# প্রথম চালু হয়  
- CLR (Common Language Runtime) আসে  
-  Windows-centric ছিল  

**C# এর জন্ম**  
**Managed code শুরু**

---

##  2007 – The Data Revolution  
###  C# 3.0 & LINQ

- LINQ যুক্ত হয়  
- Database, List, Array–এর সাথে কাজ সহজ হয়  

```csharp
var result = list.Where(x => x > 5);

```

## 2012 – The Async Era  
### C# 5.0 | async / await

- async এবং await introduce হয়  
- Asynchronous programming অনেক সহজ হয়  
- Long-running task এ app আর freeze করে না  
- Better performance
- Smooth user experience
  
```csharp
await Task.Delay(1000);
```

## 2016 – Cross-Platform Rebirth
### .NET Core 1.0

- Windows ছাড়াও Linux ও macOS এ চলে
- Open Source হিসেবে প্রকাশিত হয়
- Lightweight এবং high performance
- C# আর শুধু Windows-এর মধ্যে সীমাবদ্ধ না
- Cross-platform development শুরু হয়


## 2020+ – Unification Era
### .NET 5, 6, 7, 8

- .NET Framework এবং .NET Core একত্রিত হয়
- একটাই platform → Unified .NET
- Web, Desktop, Mobile, Cloud — সব এক জায়গা
- Modern, scalable, future-ready ecosystem

Q1. When was C# introduced?

- C# was introduced in 2002 by Microsoft.

Q2. What is LINQ?

- LINQ stands for Language Integrated Query, used to query data in a simple way.

Q3. What problem does async/await solve?

- It prevents the program from freezing during long-running tasks.

Q4. What is .NET Core?

- .NET Core is a cross-platform, open-source version of .NET.

Q5. What is meant by .NET Unification?

- Combining all .NET platforms into a single framework (.NET 5+).
